# Browsers
app was tested on chrome and firefox. 

# Installation & Run

1. npm install
2. npm start
3. run on your browser at http://localhost:3000

# More options for deployment

1. install docker
2. docker build -t playbuzz-app .
3. docker run -p 3000:3000 -d playbuzz-app

# Testing 

1. npm install mocha -g
2. npm install
3. npm test
